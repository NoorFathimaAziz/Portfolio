Overview

TeamXel is a project management tool developed while working at IBM, tailored for a client to enhance team collaboration, task tracking, and Agile workflow execution. The platform centralizes task management, timesheets, employee records, and sprint planning to digitize everyday project tracking and reporting.

Key contributions

  • Built and integrated Flask-based RESTful APIs to support real-time updates for task status, user actions, and project progress.
  • Implemented task tracking, timesheet management, employee management, and Agile sprint planning modules.
  • Designed scalable backend logic, secure endpoints, and optimized data models for performance and maintainability.
  • Enabled efficient data flow between frontend and backend to support dynamic UI interactions and seamless user experience.
  • Improved team productivity by automating project tracking and reporting processes.
